# Date and Time

JavaScript has a special object called `Date` that handles dates and times. You can use it to:
* Show the current date and time
* Store when something was created
* Calculate how much time has passed
* Work with dates in your programs

## Creating Date Objects

There are several ways to create a date. Let's start with the simplest:

**Get the current date and time:**
```js
let now = new Date();
alert( now ); // shows today's date and time
```

**Create a date from milliseconds:**
JavaScript counts time in milliseconds (thousandths of a second) since January 1, 1970. This number is called a "timestamp".

```js
// January 1, 1970 at midnight
let Jan01_1970 = new Date(0);
alert( Jan01_1970 );

// Add 24 hours (24 hours × 60 minutes × 60 seconds × 1000 milliseconds)
let Jan02_1970 = new Date(24 * 3600 * 1000);
alert( Jan02_1970 );
```

Dates before 1970 have negative timestamps:
```js
// December 31, 1969
let Dec31_1969 = new Date(-24 * 3600 * 1000);
alert( Dec31_1969 );
```

**Create a date from a string:**
```js
let date = new Date("2017-01-26");
alert(date);
```

The string gets converted automatically. The time defaults to midnight, and JavaScript adjusts it to your timezone.

**Create a date with specific parts:**
```js
new Date(year, month, date, hours, minutes, seconds, milliseconds)
```

Important things to remember:
* Year needs 4 digits (like 2024, not 24)
* Month starts at 0! January is 0, February is 1, December is 11
* Day of month is 1-31
* Hours, minutes, seconds, and milliseconds are optional (default to 0)

Examples:
```js
new Date(2011, 0, 1, 0, 0, 0, 0); // Jan 1, 2011, midnight
new Date(2011, 0, 1); // same thing (hours default to 0)

let date = new Date(2011, 0, 1, 2, 3, 4, 567);
alert( date ); // 1.01.2011, 02:03:04.567
```

## Getting Date Information

Once you have a date object, you can get information from it:

**Get the year (4 digits):**
```js
let date = new Date();
alert( date.getFullYear() ); // 2024 (or whatever year it is)
```

**Get the month (0-11):**
```js
alert( date.getMonth() ); // 0 for January, 11 for December
```

**Get the day of month (1-31):**
```js
alert( date.getDate() ); // 1 to 31
```

**Get time parts:**
```js
alert( date.getHours() );    // 0 to 23
alert( date.getMinutes() );   // 0 to 59
alert( date.getSeconds() );   // 0 to 59
alert( date.getMilliseconds() ); // 0 to 999
```

**Get the day of week (0-6):**
```js
alert( date.getDay() ); // 0 = Sunday, 1 = Monday, ..., 6 = Saturday
```

All these methods give you the time in YOUR timezone. If you want UTC time (Greenwich time), add "UTC" after "get":

```js
alert( date.getHours() );     // your local time
alert( date.getUTCHours() );  // UTC time (London time)
```

**Get the timestamp:**
```js
alert( date.getTime() ); // milliseconds since Jan 1, 1970
```

**Get timezone offset:**
```js
alert( new Date().getTimezoneOffset() ); 
// difference between UTC and your timezone in minutes
```

## Changing Date Information

You can change parts of a date:

```js
let today = new Date();

today.setHours(0); // set hour to 0 (midnight)
alert(today); // same day, but at midnight

today.setHours(0, 0, 0, 0); // set hours, minutes, seconds, milliseconds all to 0
alert(today); // exactly midnight: 00:00:00.000
```

Other methods:
* `setFullYear(year, month, date)`
* `setMonth(month, date)`
* `setDate(date)`
* `setHours(hour, min, sec, ms)`
* `setMinutes(min, sec, ms)`
* `setSeconds(sec, ms)`
* `setMilliseconds(ms)`
* `setTime(milliseconds)` - sets the whole date

Each has a UTC version too (like `setUTCHours()`).

## Automatic Date Correction

Here's something cool: JavaScript automatically fixes invalid dates!

```js
let date = new Date(2013, 0, 32); // 32nd of January? That doesn't exist!
alert(date); // JavaScript fixes it to February 1, 2013!
```

This makes it easy to add days. Want to add 2 days to a date? Just do it:

```js
let date = new Date(2016, 1, 28); // Feb 28, 2016
date.setDate(date.getDate() + 2); // add 2 days
alert( date ); // March 1, 2016 (or March 2 if it's a leap year)
```

JavaScript handles leap years and month boundaries automatically!

You can even use zero or negative numbers:
```js
let date = new Date(2016, 0, 2); // Jan 2, 2016

date.setDate(1); // set to day 1
alert( date ); // Jan 1, 2016

date.setDate(0); // day 0 means last day of previous month
alert( date ); // Dec 31, 2015
```

## Comparing Dates and Measuring Time

When you convert a date to a number, it becomes a timestamp:

```js
let date = new Date();
alert(+date); // same as date.getTime()
```

This means you can subtract dates to find the difference:

```js
let start = new Date(); // start time

// do some work
for (let i = 0; i < 100000; i++) {
  let doSomething = i * i * i;
}

let end = new Date(); // end time

alert( `The loop took ${end - start} ms` ); // difference in milliseconds
```

## Date.now() - Quick Timestamp

If you just want the current timestamp (not a full date object), use `Date.now()`:

```js
let start = Date.now(); // current timestamp

// do some work
for (let i = 0; i < 100000; i++) {
  let doSomething = i * i * i;
}

let end = Date.now(); // new timestamp

alert( `The loop took ${end - start} ms` );
```

This is faster than creating a Date object because it just gives you a number.

## Parsing Dates from Strings

You can create a date from a string using `Date.parse()`:

The format is: `YYYY-MM-DDTHH:mm:ss.sssZ`

Where:
* `YYYY-MM-DD` is the date
* `T` separates date and time
* `HH:mm:ss.sss` is the time
* `Z` means UTC timezone (or use `+05:30` for other timezones)

Shorter formats work too: `YYYY-MM-DD` or just `YYYY-MM` or `YYYY`.

```js
let ms = Date.parse('2012-01-26T13:51:50.417-07:00');
alert(ms); // timestamp in milliseconds

// Create a date from the timestamp
let date = new Date( Date.parse('2012-01-26T13:51:50.417-07:00') );
alert(date);
```

If the string format is wrong, `Date.parse()` returns `NaN`.

## Quick Recap

**Creating dates:**
* `new Date()` - current date and time
* `new Date(milliseconds)` - from timestamp
* `new Date(year, month, date, ...)` - from parts (remember: month starts at 0!)
* `new Date("string")` - from string

**Getting information:**
* `getFullYear()`, `getMonth()`, `getDate()`, `getHours()`, etc.
* `getDay()` - day of week (0 = Sunday)
* `getTime()` - get timestamp
* Add "UTC" for UTC versions: `getUTCHours()`, etc.

**Setting information:**
* `setFullYear()`, `setMonth()`, `setDate()`, `setHours()`, etc.
* `setTime(milliseconds)` - set entire date

**Important notes:**
* Months start at 0 (January = 0, December = 11)
* Days of week start at 0 (Sunday = 0, Saturday = 6)
* JavaScript automatically fixes invalid dates
* Dates can be subtracted to get time differences
* Timestamps are in milliseconds, not seconds
* Use `Date.now()` for quick current timestamp

Dates are powerful tools for working with time in your programs!
